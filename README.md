## 📖 **Project Overview**

✨ **Sundown Studio** is a **creative frontend website** designed to showcase **smooth animations, interactive elements, and responsive layouts**.  
The project demonstrates how to combine **modern web technologies** like **Lenis (for smooth scrolling)** and **Swiper.js (for sliders)** to create an **engaging user experience**.  

---

## 🚀 **Features**

- 🎨 **Creative UI/UX Design** with **HTML5 & CSS3**
- 🌀 **Lenis Integration** → Smooth scrolling effect
- 🎞️ **CSS Animations + JavaScript** → Dynamic interactivity
- 📸 **Swiper.js Slider** → Beautiful sliding galleries
- 📱 **Responsive Design** → Works on desktop, tablet & mobile
- ⚡ **Optimized Performance** → Seamless navigation & lightweight

---

## 🛠️ **Tech Stack**

- 🧩 **HTML5** → Structure & Layout  
- 🎨 **CSS3** → Styling, Animations, and Responsiveness  
- ⚡ **JavaScript (ES6+)** → Functionality & Logic  
- 🌀 **Lenis** → Smooth Scrolling Library  
- 📸 **Swiper.js** → Modern Slider/Carousel
